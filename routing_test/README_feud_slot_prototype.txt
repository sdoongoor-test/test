# Feud Go Slot Approach Comparison Prototype

This package compares two reel approaches using the same 50-spin Logic Tape.

## Open
Open `feud_slot_vs_category_prototype.html` in a browser. No server required.

## Options in the demo
- **Side-by-side**: compares both approaches on the same spin.
- **Pure Slot Symbols**: Coin Master-style 7-symbol set: Coin, Bag, Spin, Shield, Survey, Feud, Chance.
- **Category / Mode / Value**: older structured reveal approach: Category -> Mode -> Value/Tier.

## 50-spin pacing
- Full content launches: 18 / 50 (36%)
- Feud-adjacent near-misses: 7 / 50 (14%)
- Relief / utility outcomes: 25 / 50 (50%)
- Total Feud presence: 25 / 50 (50%)

## Included data
- `feud_slot_50_spin_test_data.csv`: combined 50-spin data for both approaches.
- `feud_pure_slot_50_spins.csv`: pure slot symbols only.
- `feud_category_mode_value_50_spins.csv`: category/mode/value only.
- `feud_slot_50_spin_test_data.json`: JSON version used by the HTML.

## Notes
This is a UX feel prototype. Visuals, timing, payouts, and copy are intentionally rough.
